## Getting Started

### Creating Your Account

**Step 1: Register**
- Visit https://narmirreborn.com/register
- Enter email address
- Create strong password (8+ characters, mix of letters/numbers/symbols)
- Agree to terms of service

**Step 2: Email Verification**
- Check your email for verification link
- Click link within 24 hours
- Email is now confirmed

**Step 3: Create Kingdom**
- Choose kingdom name (unique, 3-20 characters)
- Select race (human, elf, dwarf, orc)
- Agree to kingdom creation
- Your kingdom is live!

### First Login

After registration:
1. Log in with email and password
2. Complete kingdom tutorial (5-10 minutes)
3. Click "Next Turn" to start gameplay
4. Visit each panel to learn features

## Account Security

### Password Security

**Best Practices:**
- ✅ Use 12+ characters
- ✅ Mix uppercase, lowercase, numbers, symbols
- ✅ Don't use personal information (birthdate, pet names)
- ✅ Don't reuse passwords from other sites
- ✅ Use password manager (LastPass, 1Password, Bitwarden)

**Strong Password Examples:**
- ❌ `password123` (too common)
- ❌ `Sarah1994!` (uses birthdate)
- ✅ `Tr0pic@lPine#Sunset42` (random, mixed case, symbols)
- ✅ `Crimson_Phoenix_Warrior_2026` (long, memorable, unique)

### Two-Factor Authentication (2FA)

**Enable 2FA to secure your account:**

1. Go to **Settings → Security**
2. Click "Enable Two-Factor Authentication"
3. Scan QR code with authenticator app (Google Authenticator, Authy, Microsoft Authenticator)
4. Enter 6-digit code from app to confirm
5. Save backup codes in safe location

**Backup Codes:**
- Save these codes in a secure location
- Use if you lose access to authenticator app
- Each code can only be used once

### Login Security

**Warning Signs of Compromise:**
- 🚨 Unfamiliar login attempts in security log
- 🚨 Kingdom changes you didn't make
- 🚨 Unexpected troops movements or trades
- 🚨 Resources missing from treasury

**If Compromised:**
1. **Immediately change password**
2. **Enable 2FA if not already enabled**
3. **Check security log** (Settings → Security)
4. **Email support** with description of issue

### Session Management

**Manage Active Sessions:**
- Go to **Settings → Security → Active Sessions**
- View all logged-in devices/locations
- "Log out all other sessions" to force sign out everywhere
- Clear old sessions you don't recognize

## Troubleshooting

### Can't Log In

**Forgot Password?**
1. Click "Forgot Password" on login page
2. Enter your email
3. Check email for reset link
4. Create new password
5. Log in with new password

**Account Locked:**
- Locked after 5 failed login attempts
- Automatically unlocked after 30 minutes
- Or reset password to regain access immediately

### Can't Receive Verification Email

**Check spam folder first!**

If still missing:
1. Go to **Settings → Account**
2. Click "Resend Verification Email"
3. Wait up to 5 minutes
4. Check email (including spam)

**Still not working?**
- Email support with your account email
- We'll manually verify your account

### Game Lag / Performance Issues

**Try these first:**
1. Refresh browser (Ctrl+R or Cmd+R)
2. Clear browser cache (Settings → Privacy)
3. Disable animations (Settings → Display)
4. Close other browser tabs
5. Check internet connection
6. Try a different browser

**If still laggy:**
- Disable sound effects (Settings → Audio)
- Reduce graphics quality (if applicable)
- Close background applications
- Restart your computer/phone

### Sync Issues (Kingdom State Wrong)

**If your kingdom state seems incorrect:**
1. Refresh page (Ctrl+R)
2. Log out and log back in
3. Try a different browser
4. Clear browser cache

**If issue persists:**
- Email support with screenshot
- Include what you expected vs what you see
- Team will investigate and fix

### Mobile App Issues

**App not launching:**
- Restart your phone
- Reinstall app
- Check storage space (need 100MB+)
- Update iOS/Android OS

**App crashes on specific screen:**
- Try using web version instead (https://narmirreborn.com)
- Report bug via in-game bug reporter
- Include device type and OS version

## Account Deletion

**Permanently delete your account:**

1. Go to **Settings → Account**
2. Click "Delete Account"
3. Enter password to confirm
4. Kingdom archived (not playable)
5. 7-day waiting period (can cancel during this time)
6. After 7 days, account permanently deleted

**Important:**
- This is **irreversible** after 7 days
- Kingdom cannot be recovered
- Progress is lost
- You can create a new account with same email after deletion
